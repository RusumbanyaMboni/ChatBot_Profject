# Cybersecurity Awareness Chatbot

## Description
This is a C# chatbot that helps users learn about cybersecurity.

## Features
- User interaction
- Cybersecurity responses
- Voice greeting (WAV using NAudio)
- ASCII art interface
- Input validation

## Technologies Used
- C#
- .NET
- NAudio

## CI (GitHub Actions)
The project uses GitHub Actions to automatically build the application.
